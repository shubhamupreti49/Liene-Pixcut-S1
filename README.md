# 🖨️ PixCut S1 Sticker Machine — Interactive Guide
 
A clean, mobile-friendly web guide for college students to learn how to use the **Liene PixCut S1** sticker printer and cutter. Designed to be accessed via a QR code placed next to the machine.
 
---
 
## 📖 About This Project
 
This repository hosts a static GitHub Pages website that walks students through the full process of using the Liene PixCut S1 — from checking the paper to collecting their finished sticker. It was built to reduce the need for one-on-one instruction for routine use, while still directing students to a Tech Tutor when something goes wrong.
 
The guide is designed to be:
- **Scanned, not typed** — accessed via a QR code on a physical display next to the machine
- **Mobile-first** — readable on any phone screen without zooming or pinching
- **Self-contained** — no login, no app, no dependencies beyond a browser
---
 
## ✨ Features
 
- **7-step interactive accordion** — steps expand one at a time with smooth animations
- **Progress tracker** — shows how many steps a student has explored
- **Animated print-layer breakdown** — visually explains the 4-pass dye-sublimation process
- **Troubleshooting & FAQ section** — covers the most common issues (Bluetooth pairing, paper jams, app crashes, cutting errors)
- **Scroll-triggered animations** — content reveals as the user scrolls down
- **Keyboard accessible** — navigable with arrow keys for accessibility
- **No frameworks, no dependencies** — pure HTML, CSS, and vanilla JavaScript
---
 
## 🖨️ About the Machine
 
The **Liene PixCut S1** is an all-in-one sticker printer and cutter that uses thermal dye-sublimation technology to produce waterproof, fade-resistant, and scratch-resistant stickers at 300 DPI. It connects to a smartphone via Bluetooth and is controlled entirely through the **Liene Photo App** (iOS & Android).
 
Key specs:
| Spec | Detail |
|---|---|
| Print Technology | 4-Layer Thermal Dye-Sublimation |
| Resolution | 300 × 300 DPI |
| Paper Sizes | 4×6" and 4×7" only (Liene brand) |
| Connectivity | Bluetooth (iOS & Android) |
| Print + Cut Time | ~1–5 minutes per sheet |
| Auto-Cutting | AI-powered kiss-cut (0.1mm blade) |
